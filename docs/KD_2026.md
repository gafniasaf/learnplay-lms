# KD 2026 (VIG/VP) - Internal Curriculum Context

This repository includes a canonical, machine-readable representation of **KD 2026** for **Verzorging en verpleging (VIG/VP)**. The data is used for internal KD-mapping, gap analysis, and content recommendations (not for teacher UI).

## Scope
- **Basisdeel + profieldeel** for:
  - **P1** Verzorgende IG (niveau 3)
  - **P2** Mbo-Verpleegkundige (niveau 4)
- KD label used in the system: **"KD 2026"**

## Canonical Files
- **KD JSON (structured)**  
  `data/kd-2026-vigvp.json`
  - Flattenable list of KD codes (B1-K1-W1 etc.)
  - Titles, roles, and short descriptions per workprocess
- **KD Codes (flat list)**  
  `data/kd-2026-codes.txt`
  - Easy lookup for gap lists and exports
- **KD Extracted Text (raw source)**  
  `tmp/kd-2026-extracted.txt`
  - Full text extracted from the PDF, kept for traceability

## Mapping + Gap Analysis

- Script: `scripts/kd-map-curated-materials.ts`
- Default run (no DB writes):  
  `npx tsx scripts/kd-map-curated-materials.ts`
- Apply KD codes to curated materials:  
  `npx tsx scripts/kd-map-curated-materials.ts --apply`
- Output files:
  - `tmp/kd-2026-material-kd-mapping.json`
  - `tmp/kd-2026-gap-report.json`
  - `tmp/kd-2026-gap-report.csv`
